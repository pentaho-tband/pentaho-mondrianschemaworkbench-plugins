Mondrian Schema Workbench

Introduction

Welcome to the Mondrian Schema Workbench. The Mondrian Schema Workbench allows you to visually create and test Mondrian OLAP cube schemas.

See the doc directory in the Workbench folder for documentation about Workbench and Mondrian.

Prerequisites

Below are the prerequisites for running the Workbench:

  • Java 1.5
  • JDBC driver for your database in the drivers directory

To Start the Workbench...

In the directory where you installed the Workbench, run workbench.bat (for Windows) or workbench.sh (for Unix/Linux).
